## 🛠 **Cómo Compilar y Ejecutar**  

1. **Compilar un programa en C:**  
   ```bash  
   gcc programa.c -o programa  
   ```  
2. **Ejecutar:**  
   ```bash  
   ./programa  
   ```  

> **Requisitos:** GCC (GNU Compiler Collection) y sistema basado en UNIX/Linux.
